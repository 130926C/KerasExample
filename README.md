KerasExample

这是一个Keras code example自学的仓库，所有案例都来源于 [Keras code Examples](https://keras.io/examples/)，里面案例的翻译都是人工翻译而非机器翻译，同时会加入一些自己学习过程中的Tips。  

环境可以从requirement.txt进行安装，主要是以下两个：
1. tensorflow-gpu == 2.6.0
2. keras == 2.6.0
